# Docs

Team documentation for the Playwright Agentic QE Framework.

## Contents

- **[team-onboarding.html](team-onboarding.html)** — Step-by-step onboarding
  presentation: how to use the framework and the automation project to create test
  suites and test cases (intake → plan → approve → generate → run → heal).

## Viewing the presentation

Open `team-onboarding.html` directly in any browser (double-click, or right-click →
Open with). It is fully self-contained — no server or internet required.

Navigation: `←` / `→` or `Space` to move, number keys to jump to a slide, `F` for
fullscreen, and the ◐ button (top-right) toggles light/dark.
